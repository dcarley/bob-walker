# bob-walker

An alternative to [alphagov/vcloud-walker](https://github.com/alphagov/vcloud-walker). Implemented in Bash and Go and Perl.

## Usage

### Bash

```sh
./bob-walker.sh
```

### Go

```sh
go run bob-walker.go
```

### Perl

```perl
./bob-walker.pl
```

### PowerShell

```powershell
ps bob-walker.ps1
```
