# bob-walker

An alternative to [alphagov/vcloud-walker](https://github.com/alphagov/vcloud-walker). Implemented in Bash, Go, Perl and Ruby.

## Usage

### Bash

```sh
./bob-walker.sh
```

### Go

```sh
go run bob-walker.go
```

### Perl

```perl
./bob-walker.pl
```

### Ruby

```ruby
ruby bob-walker.rb
```
